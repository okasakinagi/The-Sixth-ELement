## 🗂️ 项目结构

```
The-Sixth-Element/
├── frontend/sixth_element/     # Vue 3 前端
│   ├── src/
│   │   ├── views/             # 页面组件
│   │   │   ├── UserProfileView.vue      # 个人主页（展示）
│   │   │   └── EditProfileView.vue      # 编辑信息页面
│   │   ├── components/        # 可复用组件
│   │   ├── router/            # 路由配置
│   │   └── assets/            # 静态资源
├── core/                      # Django 核心应用
├── module/survey_app/         # Django 后端
└── doc/                       # 文档
```

## 💡 技术栈
- **前端**: Vue 3, Vue Router, Vite
- **后端**: Django, Django REST Framework
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **认证**: JWT

## 📝 备注
- 设计风格：蓝白色调，年轻化、轻量化
- 用户体验：非强制填写，重引导策略
- 核心价值：精准匹配，提高问卷填写效率
